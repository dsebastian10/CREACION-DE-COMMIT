7.-en el terminal,   (' ls ' aparece el nombre del archivo a copiar)

9.-visualizar el estado (git status)
10.-muestra los cambios hechos de manera local, sin agregar al proyecto
11.-agregamos los archivos al proyecto (git add saludo.txt)
12.-Se puede modificar nuevamente, en el archivo escribir, guardar 
13.-se puede visualizar la diferencia el antes y el despues (git diff)
14.-agregamos los archivos al proyecto (git add .)
15.-Empaquetamos el proyecto con un mensaje ( git commit -m "este es mi aporte")
16.-Subimos a nuestro repositorio en la nube (git push)
